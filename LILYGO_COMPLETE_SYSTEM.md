## 15. TROUBLESHOOTING

### "No device detected"
- Check USB cable connection
- Install CH340 drivers (if needed)
- Try different USB port
- Click "🔄 Scan for Devices"

### "Payload won't upload"
- Check dev mode status
- Verify LilyGo is in bootloader mode
- Review deployment log for errors

### "Code won't compile"
- Check Arduino code syntax
- Review Viress's technical analysis
- Test simpler payload first
