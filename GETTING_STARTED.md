# Getting Started with Erryn's Soul Application

Welcome to Erryn's Soul! This document serves as a comprehensive guide to help you download, install, and use the application for the first time.

## Prerequisites
Before you begin, ensure you have the following software installed:
- [Node.js](https://nodejs.org/): Version 14 or higher (including npm)
- [Git](https://git-scm.com/): Version 2.27 or higher (for version control)

## Step 1: Downloading the Application
1. Navigate to the [Erryn's Soul GitHub repository](https://github.com/stu4331/EchoChain).
2. Click on the green **Code** button.
3. Choose one of the following options to download:  
   - Clone the repository using Git with the command:
     ```bash
     git clone https://github.com/stu4331/EchoChain.git
     ```
   - Download the ZIP file by selecting **Download ZIP** and then extracting it.

## Step 2: Installing Dependencies
1. Open your terminal or command prompt.
2. Navigate to the directory where you downloaded the repository:
   ```bash
   cd EchoChain
   ```
3. Install the required dependencies by running:
   ```bash
   npm install
   ```

## Step 3: Running the Application
1. To start the application, use the following command:
   ```bash
   npm start
   ```
2. Open your web browser and navigate to `http://localhost:3000` to access the application.

## Common Issues
- If you encounter issues during installation, please ensure that Node.js and Git are correctly installed and accessible from your command line.

## Further Resources

## Conclusion
You're all set to explore the Erryn's Soul application. Happy exploring!
